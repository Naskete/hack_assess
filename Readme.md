Method: POST
URL: 121.199.32.101:8088/:group/:id

group:组名 （front, back...
id: 组号 （1-n

```json
{
    // 打分项，user_id, group_id

    "user_id":1234,
    "group_id":"2",
}
```

```json
{
    "data": null,
    "message": "successful",
    "status": 1
}
```

```json

    "data": null,
    "message": "failed to mark",
    "status": 0
}
```
